<h1>🎯 My Hero Academia - Class 1-A Showcase</h1>

Este projeto é uma landing page interativa dedicada aos estudantes da Turma 1-A da obra My Hero Academia. A aplicação permite navegar entre as fichas técnicas detalhadas dos personagens com um design moderno, limpo e totalmente integrado à estética do anime.

<h2>📝 Sobre o Projeto</h2>

O foco principal é a apresentação dinâmica e estruturada dos heróis em treinamento da prestigiada escola U.A.:
- **Fichas Individuais Completas:** Destaque para o nome de herói, perfil de personalidade e descrição detalhada das habilidades de cada aluno.
- **Detalhamento de Individualidades:** Explicação interativa sobre os poderes (Quirks) de cada personagem, exemplificando mecânicas como o "Ácido" da Mina Ashido e as habilidades dos demais colegas de classe.
- **Alternância de Temas (Dark/Light):** A interface conta com um botão inteligente no menu superior que permite ao usuário alternar instantaneamente entre o "Tema Claro" e o "Tema Escuro", modificando o background e as cores dos textos para uma leitura mais confortável.
- **Navegação Fluida:** Através de comandos intuitivos (como o botão "Próximo"), o usuário consegue alternar o conteúdo e as imagens do card central sem a necessidade de recarregar a página.

<h2>🛠️ Tecnologias Utilizadas</h2>

HTML5: Estruturação semântica do grid, contêineres de informações e botões de controle.

CSS3: Estilização avançada com variáveis de cores para controle de temas, Flexbox/Grid para alinhamento responsivo e transições suaves.

JavaScript (ES6+): Lógica para manipulação do DOM responsável pela alternância de temas (Dark Mode / Light Mode) e controle do fluxo de exibição e paginação dos alunos.

Google Fonts: Tipografia personalizada para os títulos e descrições dos perfis.

<h2>✨ Funcionalidades</h2>

[x] Catálogo Interativo: Sistema dinâmico para navegar e descobrir as informações de todos os alunos da Turma A.

[x] Suporte a Dark Mode: Chaveamento completo de cores e fundos adaptáveis para o modo escuro.

[x] Galeria de Expressões: Exibição de artes oficiais de corpo inteiro e variações de expressões faciais para cada herói.

[x] Responsividade: Interface totalmente otimizada e adaptada para dispositivos móveis, tablets e desktops.
